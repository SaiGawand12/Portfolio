## 📥 Installation & Setup
1. **Clone the repository**
   ```bash
   git clone https://github.com/SaiGawand12/Portfolio.git
   cd Portfolio
   ```
2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```
3. **Run the development server**
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🚀 Deployment
